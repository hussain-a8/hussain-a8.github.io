CSS Positioning Cheat Sheet

This page will highlight the key CSS properties and their values!

Property Name Property Value Explanation
position relative This value allows you to position an element relative to its default static position on the web page
static The default value (it does not need to be specified)
absolute Other elements on the page will ignore the element.The element will be positioned relative to its closest positioned parent element and can positioned with offset properties
fixed Fixes the element on screen, stays in position when scrolling. It can be positioned with offset properties
sticky Keeps an element in the document flow as the user scrolls, but sticks to a specified position as the page is scrolled further
display block Block elements fill the entire width of the page by default, but their width property can also be set and their height is determined by content.
inline Inline elements cannot be altered in size with the height or width CSS properties, e.g. anchor tags
inline-block Inline-block display combines features of both inline and block elements. Inline-block elements can appear next to each other, and we can specify their dimensions using the width and height properties.
float left Floats the element as far left as possible.
right Floats the element as far right as possible.
z-index 0-10 Order of depth of elements. Determines which elements pass over what. 0 is deepest, 10 is the higher.
